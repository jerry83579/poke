## 簡介
此為前端練習作品，主要功能為：

+ 前台: 商品展示（分類、單一商品詳細介紹）、購物車功能、下單、結帳資訊、管理員登入等
+ 後台: 產品管理（新增/修改/刪除）、訂單管理、優惠券管理。

### 主要練習
+ VueCli (2) --> (3) 10/8
+ Vuex
+ VueRouter
+ 元件概念
+ Webpack
+ API操作
+ RWD
+ 手刻前台css
